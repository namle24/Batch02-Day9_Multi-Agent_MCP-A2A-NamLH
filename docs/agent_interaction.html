Cách dùng khi demo

  1. Mở file này trong browser: docs/agent_interaction.html.
  2. Chọn kịch bản câu hỏi ở dropdown phía trên.
  3. Bấm Chạy Full Stage 5 để mô phỏng luồng baseline.
  4. Bấm Chạy Optimized để mô phỏng luồng giảm latency.
  5. Mở terminal chạy command ở dưới nếu muốn show kết quả thật.
